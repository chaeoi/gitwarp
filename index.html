Pull-only registry proxy

GitWarp

快速使用

把当前站点域名加到镜像名前面，然后照常执行 Docker pull。

gitwarp.pages.dev/alpine:latest
gitwarp.pages.dev/library/nginx:latest
gitwarp.pages.dev/ghcr.io/owner/image:tag

Registry 路径

Docker Hub 默认走根路径，其他 registry 保留原始 host。

Docker Hub gitwarp.pages.dev/alpine:latest
GHCR gitwarp.pages.dev/ghcr.io/owner/image:tag
Quay gitwarp.pages.dev/quay.io/prometheus/prometheus:latest
Kubernetes gitwarp.pages.dev/registry.k8s.io/pause:3.9
MCR gitwarp.pages.dev/mcr.microsoft.com/dotnet/runtime:8.0
ECR Public gitwarp.pages.dev/public.ecr.aws/nginx/nginx:latest

Docker Hub Mirror

也可以把当前站点设置成 Docker Hub registry mirror。

{
  "registry-mirrors": ["https://gitwarp.pages.dev"]
}
